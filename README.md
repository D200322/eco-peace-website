# eco-peace-website - 2024
Environmental Pollution Project
Description

This is a simple web project created using HTML, CSS, and JavaScript.
The project explains environmental pollution, its types, causes, and effects.
It is mainly designed for learning purposes and to raise awareness about protecting the environment.

Technologies Used
HTML5-Structure of the website
CSS-Styling and responsive layou
JavaScript-Interactivity and dynamic content

How to Run
Download or clone the project.
Open the index.html file in a web browser.

Purpose
To learn basic web development
To spread awareness about environmental pollution
